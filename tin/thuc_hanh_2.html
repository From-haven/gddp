HTML là gì

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu siêu văn bản, được dùng để thiết kế nội dung trang web.

Cấu trúc của một tệp HTML là cấu trúc dạng cây của các phần tử HTML.

Link xem chi tiết về HTML tại đây: https://en.wikipedia.org/wiki/HTML

Link xem và tự học HTML tại đây:https://www.w3schools.com/html/default.asp

Giới thiệu CSS

CSS là gì

CSS (Cascading Style Sheets) là ngôn ngữ định dạng đặc biệt để mô tả cách hiển thị HTML trên trang web.

Mỗi định dạng CSS có dạng {thuộc tính: giá trị; }.

Link xem chi tiết về CSS tại đây: https://en.wikipedia.org/wiki/CSS

Link xem và tự học CSS tại đây:https://www.w3schools.com/css/default.asp
